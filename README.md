# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app) and [Ant Design] (https://ant.design/)


### The introduction for this project

1. Use ReactJS for JS library.
2. Use Jest and Enzyme for unit testing.
3. Adapt responsive web design from smallest screen to desktop screen.
4. Use React hooks for cleaner and maintainable code(reusable).
5. Use Ant-Design and Styled-Component for styling UX/UI.
6. Use MomentJS to deal with date-format.
7. For presenting better UX/UI, add search bar, login, logout function for bonus
8. Add axios interceptors for API failures and network error.

## Available Scripts

In the project directory, you can run:

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

### `yarn test`

Launches the test runner in the interactive watch mode.\
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.

### `yarn test coverage`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.
